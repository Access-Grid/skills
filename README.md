# accessgrid.com-skills

Open-source skill pack for AccessGrid integrations.

## Included Skills

- `integrate-accessgrid`: Integrate AccessGrid wallet credentials into an existing PACS with clear pathing for cloud/on-prem and scope tiers.

## Repo Layout

- `skills/<skill-name>/SKILL.md`: Skill definition and execution workflow.

## Notes

- Skill docs are written for deterministic execution: decision routing first, then task checklists, then acceptance criteria.
- Volatile implementation details should be linked to canonical docs instead of copied inline.
